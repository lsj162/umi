# `初始化`

    yarn create @umijs/umi-app

# `目录结构`

    1. 路由

        - config
            - config umi 配置
            - routes 自定义路由

        -  src
            - layouts
            - pages
            - services
            - utils

        - package
            - script 配置环境命令
            - 提交验证
            - 代码规范
